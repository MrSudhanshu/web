Snake Game

Python, Pygame

My project is a Snake game that features an awesome graphical user interface (GUI) developed using the Python programming language and the Pygame library. The game is a classic, simple and fun game where the player controls a snake which moves around a playing field while collecting food. The snake grows in length with each piece of food it collects, and the player must avoid running into the walls or the snake's own tail. The GUI of the game has been designed to be visually appealing and intuitive to use. It features colorful graphics, easy-to-use controls, and a high-score system that allows players to track their progress.

Online Food Ordering System

JAVA Full-Stack

My online food ordering system is a web-based platform that allows customers to place food orders from participating restaurants and food establishments. The system is user-friendly, easy to navigate, and offers a wide variety of food options to choose from. Customers can browse menus, place orders, and make payments all through the platform. The system will then send the order to the restaurant, which can accept or reject the order. Once the order is accepted, the restaurant will prepare the food and the customer can pick up the order or have it delivered.
